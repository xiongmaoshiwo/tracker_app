セットアップ手順
  1. 新しいGoogleスプレッドシートを作成し、名前を「StudyFlow_Backup」にします。
  2. 上部メニュー [拡張機能] → [Apps Script] を開きます。
  3. エディタの内容を全て消し、以下のコードを貼り付けます：
function doPost(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheets()[0];
  const data = JSON.parse(e.postData.contents);
  sheet.getRange(1, 1).setValue(JSON.stringify(data));
  return ContentService.createTextOutput(JSON.stringify({status:"ok", data: data}))
    .setMimeType(ContentService.MimeType.JSON);
}
function doGet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheets()[0];
  const val = sheet.getRange(1, 1).getValue();
  return ContentService.createTextOutput(val || '{"records":[]}')
    .setMimeType(ContentService.MimeType.JSON);
}
  1. 右上の [デプロイ] → [新しいデプロイ] を選択。
  2. 種類：ウェブアプリ、実行ユーザー：自分、アクセスできるユーザー：全員。
  3. [デプロイ] を実行し、発行された ウェブアプリのURL をコピーしてアプリに貼り付けます。
